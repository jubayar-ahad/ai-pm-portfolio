# Career Transition Package

This folder contains the AI PM transition materials created from Jubayar Ahad's current product-management background. The package is intentionally framed as a transition into AI-enabled product work, not as a claim of prior production AI/ML model ownership.

## Files

- [AI PM Transition Plan](AI_PM_Transition_Plan.md): positioning strategy, best-fit AI PM lanes, 30/90-day roadmap, skills gap map, and weekly job-search routine.
- [Resume Positioning](Resume_Positioning.md): headline options, summary options, AI-safe competency language, role-by-role bullet rewrites, and guidance on resume status/date consistency.
- [Portfolio Project Ideas](Portfolio_Project_Ideas.md): AI PM portfolio concepts tied to retail, eCommerce, mobile, checkout, fulfillment, enterprise workflows, payments, risk, and analytics.
- [Interview Story Bank](Interview_Story_Bank.md): STAR-style stories mapped to AI PM interview themes without inventing facts.
- [LinkedIn Profile Draft](LinkedIn_Profile_Draft.md): headline variants, About section draft, Featured portfolio recommendations, experience repositioning notes, and outreach blurbs.

## Positioning Rule

Use "AI-enabled" and "AI-focused transition" when describing this work. Avoid saying "owned AI/ML products," "managed model training," or "led LLM strategy" unless a future role or shipped portfolio project directly supports that claim.
