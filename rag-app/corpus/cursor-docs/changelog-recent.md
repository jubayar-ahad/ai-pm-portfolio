# Cursor Changelog — Recent Releases

A condensed view of the most recent entries from `cursor.com/changelog`, retrieved 2026-05-17.

## 3.4 — May 13, 2026 — Development environments for cloud agents

Cloud agents now support multi-repo environments and improved Dockerfile-based configuration. The release includes build secrets for secure package registry access, with "70% faster" cached builds. Agents provide interactive setup with credential validation and version history with rollback capabilities. Security features include audit logs and environment-scoped secrets.

## May 11, 2026 — Cursor in Microsoft Teams

Cursor integrates with Microsoft Teams. Users can mention @Cursor in channels "to delegate a task to a cloud agent or pull information from Cursor into Teams." The system automatically selects repositories and models, reads thread context, and creates pull requests for team review.

## May 11, 2026 — Bugbot Effort Levels

Teams and Individual plan users can customize Bugbot's PR review effort across three configurations: Default (optimized for speed), High (more reasoning), and Custom (natural language instructions). "With default effort, Bugbot finds 0.7 bugs per run, on average." High effort averages 0.95 bugs per run.

## 3.3 — May 7, 2026 — PR Review, Build Plan in Parallel, and Split PRs

New PR review interface includes Reviews, Commits, and Changes tabs with inline threads and file navigation. "Build in Parallel" executes independent plan tasks simultaneously using async subagents. Users can split changes into logical pull requests and pin frequently used skills as quick-action buttons.
